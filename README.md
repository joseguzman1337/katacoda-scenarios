## Security

This repository follows the [OpenSSF Best Practices
guidelines](https://bestpractices.coreinfrastructure.org/projects/6430). See
[`SECURITY.md`](./SECURITY.md) for the supported-version matrix, the
vulnerability-reporting process, and our coordinated-disclosure safe harbor.
Automated security tooling (Scorecard, Dependabot alerts, dependency-review,
branch protection) is configured under `.github/`.
